1. Change Management

We are using Jira to track and see the changes and requirements.
Steps to do a change request:

  1. You have to be logged in to make a request.
  2. To make the request, you can push the button “Create Issue”.
  3. Our project “CLOUD” has to be selected to make a dedicaded request.
  4. The more precise the description of the issue is, the better we can response to it.
  5. Since the user who created the issue is registered, we can contact him directly in case of questions and inexactnesses.
